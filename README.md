# 🚀 iRede E-commerce - Hackathon Fullstack

Projeto desenvolvido para o Hackathon da Residência em TIC 12. Foco em interface fluida, responsividade e usabilidade.

## 📋 MVP Realizado 
- [x] **Página Inicial**: Banner dinâmico e vitrine de produtos.
- [x] **Listagem e Detalhes**: Navegação dinâmica com IDs e dados mockados.
- [x] **Carrinho**: Controle de quantidades e cálculo de preço total.
- [x] **Checkout**: Fluxo simplificado de confirmação de compra.
- [x] **Admin**: Dashboard com DataTable para gestão de estoque.

## 💻 Stack Técnica
- **Framework**: Vue 3 (Options API) + TypeScript.
- **UI**: PrimeVue v4 com Tema Aura.
- **Estilo**: Tailwind CSS v3.4 (Configurado para máxima compatibilidade).
- **Rotas**: Vue Router 4 (Com Navigation Guards).

## ⚙️ Instalação e Execução
1. `npm install`
2. `npm run dev`
